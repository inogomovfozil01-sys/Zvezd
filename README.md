# Giveaway Bot

Telegram-бот для розыгрышей с одним администратором:

- админ создает розыгрыш через `/start`
- бот публикует анонс в канал
- участники заходят в бота и нажимают кнопку участия
- после набора 10 участников админ запускает голосование
- после закрытия опроса бот определяет победителя и уведомляет админа

## Запуск

1. Установите зависимости:

```powershell
npm install
```

2. Создайте `.env` на основе `.env.example`

3. Запустите бота:

```powershell
npm start
```

## Переменные окружения

- `BOT_TOKEN` - токен Telegram-бота
- `ADMIN_ID` - Telegram ID администратора
- `CHANNEL_ID` - username канала или ID вида `-100...`
- `BOT_USERNAME` - username бота, например `@my_bot`
- `VOTE_URL` - ссылка, которую бот вставляет в сообщение про голосование

## Важно

- Бот использует long polling.
- Состояние розыгрыша сохраняется в `data/state.json`.
- Голосование завершается, когда сам опрос закрыт в Telegram. После закрытия бот определяет победителя по числу голосов.
